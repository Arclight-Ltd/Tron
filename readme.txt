1、由于qemu对鼠标的解码较差，在qemu中使用鼠标拖动窗口将会造成一定程度的卡顿。若要获得良好的体验可将该文件夹下的Tron.img文件加载到VMware虚拟机中运行。
2、系统操作指南：
①Console为命令行窗口，目前其支持关键词实现指令触发，即输入语句只要包含（不区分大小写）：
*whatareyou―――――――提供系统介绍信息
*name――――――――――提供系统名字
*mem―――――――――― 提供目前系统内存情况
*cls―――――――――― 清空当前命令行窗口文字
*file――――――――――显示系统文件信息
*run+文件名―――――――运行文件。目前只做了一个APP1文件
*cmd―――――――――― 新建一个命令行窗口
*exit――――――――――推出当前命令行窗口
②按下shift键实现大小写切换
③按下ctrl键新建一个命令行窗口
④选中的窗口顶端会变为黑色，未选中的窗口顶端为灰色
⑤点击窗口右上方的叉可以关闭窗口
⑥可按下win键从qemu释放鼠标
3、Tron内为操作系统源码，所有的C语言文件为自主编写（除部分算法）。部分汇编文件参考了作者源码。
